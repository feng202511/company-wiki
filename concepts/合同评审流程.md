# 合同评审流程

## 定义
合同签署前，由法务、财务、业务等多部门联合审查的标准化流程。

## 标准评审流程
```
起草 → 业务初审 → 法务审查 → 财务审查 → 审批签署 → 归档
```

## 各部门评审要点
- **业务部** — 条款完整性、交付物定义
- **法务** — 法律风险、违约责任、争议解决
- **财务** — 付款条件、发票、税务、预算
- **管理层** — 根据金额分级审批
