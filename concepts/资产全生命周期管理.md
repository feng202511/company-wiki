# 资产全生命周期管理

## 定义
从资产需求提出到报废处置的全过程管理，确保资产安全、高效利用。

## 全生命周期流程
```
申购 → 采购 → 验收 → 入库（资产卡片）→ 领用 → 使用维护 → 盘点 → 维修/调拨 → 报废/处置
```

## 关键管控点
- **资产卡片** — 每项资产唯一编号，记录位置、责任人、使用状态
- **定期盘点** — 年度全面盘点 + 季度抽盘
- **维修/调拨** — 需审批留痕，记录变更历史
- **报废处置** — 技术鉴定→审批→残值回收→销账
