# 供应商管理

## 定义
对供应商从准入、合作、评估到退出的全生命周期管理。

## 全生命周期
```
供应商寻源 → 资质审核 → 样品验证 → 准入 → 合作 → 绩效评价 → 淘汰/升级
```

## 核心管控点
- **准入** — 营业执照、资质证书、行业口碑、财务健康度
- **分级** — 战略供应商、普通供应商、试用供应商
- **招标/比价** — 按金额门槛执行不同采购方式
- **绩效评价** — 质量、交期、价格、服务四维度打分
- **淘汰机制** — 连续不达标或出现重大质量问题
