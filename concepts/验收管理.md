# 验收管理

## 定义
对不同业务场景下的交付物进行检验、确认、接收的标准化流程。

## 验收类型
| 验收类型 | 适用场景 | 制度 |
|---------|---------|------|
| 来料检验（IQC） | 采购物料入库前 | [[供应链管理制度]] |
| 制程检验（IPQC） | 生产过程中 | [[生产管理制度]] |
| 成品检验（OQC） | 成品出厂前 | [[生产管理制度]] |
| 资产验收 | 固定资产购入后 | [[固定资产管理制度]] |
| 项目验收 | 项目交付给客户 | [[重点客户类项目管理制度]] |
| 样机验收 | EVT/DVT/PVT各阶段 | [[研发管理制度]] |

## 通用验收流程
1. **发起** — 交付方提交验收申请
2. **检验** — 使用标准/规格书逐项检查
3. **记录** — 记录检验结果，不合格项标记
4. **确认** — 合格签收 / 不合格退回整改
5. **归档** — 验收报告、检验记录存档
