# 重点客户类项目管理制度

**版本：** V1.0 | **发布：** 2026年6月 | **制定部门：** 量子研发部/软件研发部

## 特点
针对重点客户的定制化项目，区别于标准产品开发。

## 核心流程
**客户需求分析 → 方案设计 → 报价/投标 → 合同签订 → 项目执行 → 验收交付 → 售后维护**

## 重点管控
- 客户需求变更管理
- 项目里程碑管理
- 定制化开发与标准产品平衡
- 验收标准与付款条件
- 客户满意度评价
