# 债权融资管理制度

**版本：** V1.0 | **发布：** 2026年6月 | **制定部门：** 财务部

## 范围
银行借款、信托、融资租赁、保理、票据贴现、债券、资产证券化、非标融资、股东及第三方借款

## 四大原则
**适度性** → **安全性** → **低成本** → **合法性**

## 管控红线
- 分子公司融资必须母公司预审 + CEO办审批
- 严禁擅自对外融资
